dronmakr is a local Python toolkit for sample generation and drum workflows. Generate ideas, edit audio, organize your library, and package exports, all from this browser.

auditionr

Audition exports, apply effects, trim and process samples, save or archive.

beatbuildr

Build drum patterns with a random kit, preview, and save patterns to config.

collections

Browse saved samples, filter your library, and package selected files for export.

folysplitr

Record, slice, edit and sort your own original audio into sample folders.
