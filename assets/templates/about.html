About dronmakr

dronmakr is a local Python toolkit for sample generation, editing, and packaging. Build drones and beats, audition and process audio, organize saved work, and export packs—all from your browser with a Flask UI backed by your own files on disk.

This release is version {{ version }}. Sources and updates live on GitHub; thanks for using the project.
