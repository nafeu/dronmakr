dronmakr for Windows
====================

Install
-------
1. Unzip this archive.
2. Open the nsis folder and run dronmakr_*_x64-setup.exe.
3. Launch dronmakr from the Start menu.
4. On first launch, choose where to store your dronmakr-files folder (generated audio, presets, config).

Other folders in this zip (for example the application bundle) are build outputs. You only
need the *-setup.exe installer to install the app.

Updates
-------
Download the latest release from GitHub and run the new installer over your existing install.
